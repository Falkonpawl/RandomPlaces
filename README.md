# RandomPlaces

Веб-приложение для случайного распределения мест с учетом предпочтений участников.

## Описание

Приложение генерирует случайное распределение мест на основе предпочтений участников, используя систему взвешенных коэффициентов. Алгоритм учитывает желаемые места каждого участника и присваивает им коэффициенты вероятности.

## Как это работает

1. **100% распределение**: Участники, которые единогласно выбрали одно и то же место, автоматически получают это место.

2. **Взвешенный случайный выбор**: Для оставшихся мест используется система коэффициентов:
   - **10 баллов** — если желаемое место совпадает с доступным
   - **5 баллов** — если желаемое место отличается на ±1 от доступного
   - **1 балл** — для всех остальных случаев

3. **Вероятностный выбор**: Чем выше коэффициент участника, тем больше вероятность получить место.

## Использование

Введите данные в формате: `Имя НомерМеста` (каждая строка — отдельный участник)

Пример:
```
Иван 1
Мария 2
Петр 1
Анна 3
```
